# 30 MVPs in 30 Days

Building 30 minimum viable products in 30 days — powered by Hermes Agent, documented faceless across YouTube, TikTok/Reels, and X.

## Goal
Ship real products fast. Document the journey publicly. Find winners.

## Rules
- Every MVP must be **live** (Vercel), **stored** (GitHub), and **documented** (social) to count as complete
- Max 4 hours per build
- Monorepo structure — one folder per MVP
- No MVP stays local

## Structure
```
/mvp-01
/mvp-02
...
/mvp-30
/shared         ← reusable components (optional)
tracker.md      ← status of all 30 builds
TEMPLATE.md     ← README template for each MVP
```

## Deployment Stack
| Layer | Tool |
|-------|------|
| Live hosting | Vercel |
| Digital products | Gumroad |
| Code | GitHub (this repo) |
| Content | YouTube · TikTok/Reels · X |
